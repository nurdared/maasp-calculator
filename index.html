MAASP Calculator

MAASP Calculator is a calculation tool used to run up MAASP (Maximum Allowable Annulus Surface Pressure) calculations at any one time for several points, providing the details of the MAASP value for the current status of the well using the derated wellhead pressure resistance and the wall thickness loss in any string and identifying the annulus element with the minimum pressure rating, i.e. the component most likely to fail first.
The technical background to the MAASP Calculator calculation is embodied in ISO16530-1 (Annex K).

What is the MAASP?

Previous Next

The Maximum Allowable Annulus Surface Pressure (MAASP) is the greatest pressure that an annulus can contain, as measured at the wellhead, without compromising the integrity of any element of that annulus, including any exposed openhole formations. The MAASP is required for each annulus of the well and should be documented. For operational wells, MAASP needs to be recalculated if there are changes in the following:

  • Well barrier elements acceptance criteria
  • Service type
  • Annulus fluid density
  • Tubing/Casing wall thickness loss
  • Reservoir pressures outside the original load case calculation

Key benefits of calculator

Previous Next
  • MAASP calculator allows users to test annuli configurations and equipment to help program and provide safer well operations in any annulus conditions
  • Mitigates risk of reaching annulus loading conditions that can surpass the available pressure capacity of the annulus components
  • Save file feature allows users to quickly adjust for differences from well to well
  • Calculations reflect the industry best practice and meet ISO16530-2 Annex K methodology
